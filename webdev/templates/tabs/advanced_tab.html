UNIFAC Input Parameters

It is highly recommended that you keep track of required inputs in a spreadsheet or similar application and copy and paste the values here.

Group contribution matrix instructions
▼

Each row represents one UNIFAC structural group (CH3, ACH, H2O, etc.).

Each column represents one specific component in the reacting mixture, labeled according to your previously selected reaction type.

The value in each cell represents the number of times a given UNIFAC group appears in the specified molecule.

Only unsigned integer values are valid (0, 1, 2, ...).

Group parameters instructions
▼

Each row represents one UNIFAC structural group (CH3, ACH, H2O, etc.).

There are two columns corresponding to parameters Rk and Qk

The number of rows in this table is linked to the number of rows in the prior table.

Energy of interaction matrix instructions
▼

This is a square matrix with M rows and columns, where M is the total number of UNIFAC structural groups.

Each cell contains the interaction parameter value between group i and group j, denoted as a(i, j).

Note that this matrix is typically not symmetric and a(i, j) will usually be different from a(j, i).
